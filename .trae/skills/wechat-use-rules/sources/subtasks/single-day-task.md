# 单天任务执行规范

> ⚠️ **本文档仅供被派发的子任务/子Agent使用。**
> 主Agent（负责编排和合并的那一方）禁止阅读本文档。

---

## 你的角色

你是一个单天数据提取执行者。你只负责处理**一天**的数据，输出一份 `extract_{date}.md`。
不派发子任务，不读取其他天的文件，不做合并。

---

## 输入参数

派发方会给你以下参数：
- `wxid`：目标联系人/群聊的 wxid
- `date`：处理的日期（格式 YYYY-MM-DD）
- `task_id`：主任务ID（用于定位 workspace 路径）
- `output_path`：消息 JSON 的完整输出路径
- `extract_path`：本次产物的写入路径

---

## 执行步骤（严格按序，不得跳过任何一步）

### 第1步：拉取当天数据

调用 `get_messages`，参数：
- `wxid` = 输入参数中的 wxid
- `start_time` = `{date} 00:00:00`
- `end_time` = `{date} 23:59:59`
- `output_path` = 输入参数中的 output_path

### 第2步：验证数据参数

```
python analyzer/verify.py arg {output_path}
```

- 返回码非 0 → **立即停止**，上报失败原因，不执行后续步骤
- JSON 超1天会被自动删除，上报后等待重新派发

### 第3步：运行预处理脚本

```
python analyzer/stats.py {output_path}
```

输出 `stats_{date}.json`（与 output_path 同目录）

> `stats.py` 只计算数字，**不产生任何分析结论**。数字是后续步骤的量化基准。

### 第4步：读取消息原文

**完整读取 `{output_path}`（messages JSON 原文），逐条过一遍所有消息。**

> ⛔ 不得跳过此步、不得只读 stats 就进行分析。
> stats 只是数字背书，所有定性判断来源于原文。

### 第5步：语义分析，写入工作区

读完原文后，执行以下分析，结果写入 `{extract_path}`：

#### 5.1 准备数字基准
从 `stats_{date}.json` 中记录：
- 日均消息数、当天消息数
- 活跃时段（小时分布）
- 媒体类型比例（文字/图片/表情/语音）

#### 5.2 异常识别
对照基准，找出当天偏离基线的数据点：
- 消息量显著高于/低于日均（**用数字说话**：不说"很多"，说"32条，是日均的2.8倍"）
- 活跃时段异常（平时不活跃的时段突然活跃）
- 媒体类型比例偏移
- 回复延迟异常

#### 5.3 特质线索归纳
结合消息原文 + 异常数据点，判断是否体现某个性格特质：
- **必须有原文依据**（引用具体消息行为）
- **必须有量化支撑**（用 stats 数字衡量偏离幅度）
- ⭐ 强度由偏离基线的幅度决定

#### 5.4 风格线索归纳
同上，针对发言风格（用词、句式、节奏、表情习惯等）

#### 5.5 关系线索
- 从原文中识别互动对象和互动性质（友善/摩擦/拉拢/回避）
- stats 给出互动频次，原文判断互动性质

#### 5.6 里程碑线索
**判断标准（两个条件同时满足才写）：**
1. 当天发生了单次事件（不是日常互动）
2. 该事件对双方关系有实质性影响

> 若无符合条件的事件，此节留空，不要随意填写。

#### 5.7 解读线索
归纳行为模式：
- 特定触发条件 → 特定反应（"每次被反驳后必然出现X行为"）
- 话语背后的隐含意图

### 第6步：验证产物格式

```
python analyzer/verify.py extract {extract_path}
```

- 返回码非 0 → **修复后重新验证，不得跳过**
- 验证通过后上报：**完成**

---

## extract_{date}.md 格式

```markdown
# extract | {wxid} | {date}

## 特质线索
- {特质名} ⭐⭐⭐ [{来源wxid}]：具体行为，"发了32条，是日均的2.8倍"（{date}）
- {特质名} ⭐⭐  [{来源wxid}]：具体行为（{date}）

## 风格线索
- {风格名} ⭐⭐⭐ [{来源wxid}]：具体样本，附量化数据（{date}）

## 关系线索
- [{wxid_a}] 和 [{wxid_b}] 发生冲突，最终互相撤回
- [{wxid_c}] 主动关心 [{wxid_a}]，连续@3次

## 里程碑线索
> 如果当天发生了对双方关系有实质影响的事件才填，否则留空
- [{wxid}] {YYYY-MM-DD} 发生了一场公开争吵；影响：关系变化/行为改变

## 解读线索
- [{wxid_a}] 说"随便"后沉默2小时，疑似掩盖真实想法
```
