# 群整体画像规范

本文件定义群整体画像的结构。由 `group-builder.md` 在第三阶段（可选）生成。

## 存储位置

```
experience/groups/{group_wxid}.md
```

## 建设前提

群整体画像基于已完成的成员画像生成，必须先完成目标成员的 `experience/members/{wxid}.md` 建设，再汇总生成本文件。

## 文件结构

```markdown
# {群名}（{group_wxid}）

## 基础信息
- 类型: 同学 / 工作 / 兴趣 / 其他
- 成员数: xxx
- 活跃度: 日均 xx 条，高峰时段 xx:00-xx:00

## 核心成员
| 昵称 | wxid | 消息数 | 群内角色 | 核心标签 |
|------|------|--------|----------|----------|
| xxx  | {wxid} | 312  | 活跃者   | 毒舌、话唠 |

## 群聊风格
- 整体氛围描述

## 小团体
- [{wxid_a}] + [{wxid_b}]：互动最频繁，xx 次，话题集中在 xxx

## 高频话题
- xxx（占比 xx%）

## 群规/潜规则
- 隐性规则描述

## 更新记录
- 2026-04-19 首次建档，基于 {date_range} 数据
```
